# 📝 NoteVault

A modern React Notes App with color-coded notes, search, edit/delete, and dark mode.

## 🚀 Features
- Add / Edit / Delete Notes
- Color-coded notes 🎨
- Search functionality 🔍
- Dark Mode 🌙
- LocalStorage support 💾

## 🛠️ Tech Stack
- React
- Vite
- JavaScript
- CSS
